## 페이스북 메신저를 이용한 부산대학교 실시간 출석 체크 및 강의 관리 챗봇

- 페이스북 메신저의 웹뷰 기능 업데이트됨.
- html2pdf 라이브러리를 이용해 자신이 생성한 시간표를 pdf 파일로 내보내기 기능 추가.
- snoopy class를 사용하여 대학 웹사이트에서 실시간으로 강의 목록 크롤링 가능.
